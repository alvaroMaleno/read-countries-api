# read-stock-api

ReadOnly stocks api

### What is this repository for? ###

* API to get static stocks information
* Version 0

### Dependencies ###

* pip install flask
* pip install python-dotenv
* pip install psycopg2-binary

### How do I get set up? ###

* python -m flask run
